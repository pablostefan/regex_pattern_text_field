# Example App

This folder contains a runnable Flutter app demonstrating `regex_pattern_text_field`.

## What this example shows

- Real-time regex highlighting while typing.
- Custom style definitions with typed metadata.
- Optional built-in default pattern set.
- Callback behavior for `onMatch`, `onNonMatch`, `onChanged`, and `onSubmitted`.
- Live list of detected matches.

## Run

```bash
flutter pub get
flutter run
```

## Notes

- The app entry point is `example/lib/main.dart`.
- You can toggle default patterns from inside the UI.
- Use the "Insert sample" button for quick manual testing.
